# TP1 — Installation et configuration (Correction / Guide)

## Installer Python 3.12
- Télécharger Python 3.12 depuis le site officiel.
- Pendant l'installation (Windows) : **cocher "Add Python to PATH"**.
- Vérifier :
```bash
python --version
```

## Installer VS Code
- Installer VS Code.
- Optionnel (Windows) : cocher **Add to PATH**.

## Configurer VS Code pour Python
- Installer l'extension **Python (Microsoft)**.
- Sélectionner l'interpréteur : `Python: Select Interpreter` → Python 3.12
- Vérifier dans le terminal intégré :
```bash
python --version
```

## Premier programme
Créer `premier_prog.py` :
```python
print("Bonjour, bienvenue en Python!")
```
Exécuter :
```bash
python premier_prog.py
```
